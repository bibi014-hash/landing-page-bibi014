POLÍTICA DE PRIVACIDADE DA BIBI 014 PASSAGEIROS

Esta Política de Privacidade descreve como a **Bibi 014** coleta, usa, armazena e compartilha seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

1. DADOS COLETADOS

Coletamos os dados necessários para o funcionamento e segurança do serviço, incluindo:

  • Dados de Cadastro: Nome completo, CPF, e-mail, número de telefone e data de nascimento.
  • Dados de Localização: Coletada durante o uso ativo do aplicativo para viabilizar o serviço de transporte.
  • Dados de Uso: IP, tipo de dispositivo, tempo gasto no app e informações sobre as transações.
  • Dados de Pagamento: Informações de cartão de crédito e histórico de transações (processados por terceiros seguros).

2. FINALIDADE DA COLETA DE DADOS

Os dados são utilizados estritamente para:

  • **Execução do Contrato:** Intermediar a solicitação e execução do serviço de transporte (localizar motoristas, traçar rotas).
  • **Segurança:** Prevenir fraudes e garantir a segurança do Usuário e do Motorista Parceiro.
  • **Comunicação:** Enviar notificações, suporte e, mediante consentimento, informações promocionais.
  • **Melhoria e Cumprimento Legal:** Análise de desempenho e atendimento a requisições judiciais.

3. COMPARTILHAMENTO DE DADOS

Seus dados são compartilhados apenas para as seguintes finalidades:

  • **Com Motoristas Parceiros:** Seu nome, foto, localização de partida e destino (apenas o essencial para a corrida).
  • **Com Prestadores de Serviço:** Empresas de pagamento, hospedagem e análise de dados (com garantia de segurança).
  • **Com Autoridades:** Mediante ordem judicial ou obrigação legal.

4. DIREITOS DO TITULAR (LGPD)

O Usuário tem direitos assegurados pela LGPD, incluindo o acesso, correção, eliminação de dados desnecessários e a revogação do consentimento.

Para exercer qualquer um desses direitos, o Usuário deve entrar em contato pelo e-mail: bibimobibarrabonita@gmail.com

5. ARMAZENAMENTO E SEGURANÇA

Armazenamos seus dados pelo tempo necessário, adotando medidas técnicas e administrativas (criptografia, firewalls) para protegê-los contra acesso não autorizado, perda ou alteração.

6. CONTATO

Para dúvidas sobre esta Política, entre em contato com nosso Encarregado de Dados (DPO) através do e-mail: bibimobibarrabonita@gmail.com.
